SMK AL FALAH TANJUNGJAYA

1. Refleksi Pemahaman Materi

Deskripsikan seberapa jauh kalian memahami materi mengenai konfigurasi dasar mikrotik?

2. Analisis Peran Mikrotik dalam Jaringan Sekolah

Bayangkan sekolahmu ingin membangun jaringan komputer menggunakan router Mikrotik. Jelaskan:

  • Mengapa Mikrotik dipilih sebagai perangkat utama,
  • Fungsi dasar Mikrotik dalam mengatur lalu lintas data jaringan sekolah,
  • Keuntungan yang didapat sekolah jika konfigurasi Mikrotik dilakukan dengan benar.

3. Konfigurasi IP Address Dasar

Kamu baru saja membeli router Mikrotik baru untuk laboratorium komputer. Jelaskan langkah-langkah dasar yang harus dilakukan untuk:

  • Mengatur IP Address pada interface Mikrotik,
  • Memastikan koneksi Mikrotik ke internet,
  • Menghubungkan jaringan lokal (LAN) ke Mikrotik.

4. Analisis DHCP Server Mikrotik

Mikrotik dapat digunakan sebagai DHCP Server. Menurut pendapatmu:

  • Apa keuntungan menggunakan DHCP dibanding IP statis untuk jaringan sekolah,
  • Bagaimana langkah-langkah dasar mengaktifkan DHCP Server di Mikrotik,
  • Dampak yang terjadi jika DHCP tidak dikonfigurasi dengan benar.

5. Studi Kasus Konfigurasi Gateway & NAT

Sekolahmu memiliki satu koneksi internet yang ingin dibagi ke seluruh komputer lab menggunakan Mikrotik. Jelaskan langkah-langkah konfigurasi:

  • Menetapkan gateway Mikrotik,
  • Mengaktifkan NAT (masquerade) agar semua komputer lab dapat mengakses internet,
  • Cara mengecek apakah konfigurasi berhasil.
