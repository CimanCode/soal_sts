These questions may seem difficult, but trust me, nothing is more powerful than your mind. Work calmly. I'm not waiting for perfect results, I'm just waiting for your smile after the struggle.

1. Refleksi Pemahaman Materi

Deskripsikan seberapa jauh kalian memahami materi mengenai website, HTML, dan CSS?

2. Struktur Dasar Website

Jelaskan menurut pendapatmu:

  • Apa perbedaan utama HTML dan CSS dalam pembuatan website?
  • Mengapa keduanya penting digunakan bersama?
  • Berikan contoh sederhana penerapan HTML dan CSS dalam satu halaman website.

3. Analisis Elemen HTML

Bayangkan kamu diminta membuat halaman profil sederhana menggunakan HTML. Sebutkan minimal lima elemen HTML yang akan kamu gunakan, jelaskan fungsi masing-masing, dan mengapa elemen itu penting.

4. Peran CSS dalam Desain

Jelaskan dengan bahasamu sendiri bagaimana CSS dapat memengaruhi tampilan dan pengalaman pengguna pada sebuah website. Beri contoh situasi di mana pemakaian CSS yang tepat bisa meningkatkan kualitas website.

5. Memahami Struktur dan Layout

Sebuah halaman web terdiri atas header, menu navigasi, konten utama, dan footer. Analisislah bagaimana kamu akan menyusun struktur HTML dan CSS agar halaman tersebut responsif dan mudah diakses di berbagai perangkat. Jelaskan secara deskriptif.

6. Studi Kasus Desain Web

Kamu diminta memperbaiki tampilan website sekolah yang warnanya tidak serasi dan teksnya sulit dibaca. Jelaskan langkah-langkah analisis dan perbaikan yang akan kamu lakukan menggunakan CSS agar website menjadi lebih menarik dan ramah pengguna.
