1. Refleksi Pemahaman Materi

Deskripsikan seberapa jauh kalian memahami materi mengenai website, HTML, dan CSS?

2. Konsep Website Modern

Jelaskan menurut pendapatmu:

  • Apa perbedaan utama antara website statis dan website dinamis?
  • Mengapa HTML, CSS, dan framework CSS menjadi dasar penting dalam pengembangan website modern?
  • Berikan satu contoh website sederhana yang menurutmu memanfaatkan ketiganya dengan baik.

3. Analisis Struktur HTML

Bayangkan kamu diminta membuat halaman profil perusahaan menggunakan HTML. Jelaskan:

  • Elemen-elemen HTML apa saja yang wajib digunakan untuk membuat struktur dasar halaman (minimal lima elemen),
  • Fungsi masing-masing elemen tersebut,
  • Mengapa penting menggunakan elemen semantik dalam HTML5.

4. Peran CSS dalam Desain Website

Jelaskan bagaimana CSS memengaruhi tampilan dan pengalaman pengguna pada sebuah website.

  • Berikan contoh pengaturan CSS (warna, font, layout) yang dapat meningkatkan kenyamanan pengunjung.
  • Jelaskan dampak negatif jika CSS digunakan berlebihan atau tidak konsisten.

5. Analisis Framework CSS

Framework CSS seperti Bootstrap atau Tailwind CSS sering digunakan dalam pengembangan web. Jelaskan

  • Apa keuntungan utama menggunakan framework CSS dibanding menulis CSS dari nol,
  • Sebutkan dua fitur yang menurutmu paling membantu pada framework CSS dan jelaskan alasannya,
  • Situasi kapan sebaiknya tidak menggunakan framework CSS.

6. Studi Kasus Responsive Design

Sekolahmu meminta kamu membuat website informasi sekolah yang responsif di berbagai perangkat. Analisislah:

  • Bagaimana kamu akan menyusun struktur HTML agar mudah diatur CSS-nya,
  • Bagaimana kamu akan memanfaatkan framework CSS untuk membuat tampilan responsif,
  • Apa yang akan kamu lakukan untuk menguji tampilan website di berbagai ukuran layar.
