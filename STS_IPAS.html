6. Studi Kasus Pembuatan Sabun Alami

Siswa kelas X SMK sedang melaksanakan proyek membuat sabun alami sederhana dengan memanfaatkan bahan-bahan yang mudah ditemukan di rumah, seperti minyak kelapa, soda api (NaOH), air, serta tambahan jeruk nipis dan daun pandan untuk aroma alami. Pada saat praktikum, ada kelompok yang terburu-buru mencampurkan soda api ke dalam minyak panas, sehingga adonan sabun cepat menggumpal, keras, dan baunya menyengat. Sementara itu, kelompok lain mengikuti langkah dengan benar, yaitu melarutkan soda api ke dalam air dingin terlebih dahulu, lalu mencampurkannya dengan minyak kelapa pada suhu yang tepat. Hasilnya, sabun yang mereka buat halus, wangi, dan dapat digunakan untuk mencuci tangan maupun membersihkan lantai.
